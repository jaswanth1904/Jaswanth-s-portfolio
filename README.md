# Jaswanth's Personal Developer Portfolio 🚀

A premium, highly-interactive, and fully responsive personal developer portfolio built to showcase modern web development skills, feature-rich projects, and professional experience.

## ✨ Features

- **Modern & Premium UI**: Custom design with glassmorphism touches, elegant typography, and carefully chosen dark/light color palettes.
- **Dynamic Theming**: Seamless Light & Dark mode toggle using CSS Variables (`--minimal-white`, `--minimal-bg`, etc.) and Tailwind's `darkMode: 'class'`.
- **Interactive Cursor Trail**: A custom, vibrant, multi-colored particle trailing cursor effect built with Canvas and Framer Motion.
- **Responsive Layouts**: Designed to be pixel-perfect on both mobile and desktop screens.
- **Dynamic Project Showcase**: A robust modals system loading real-time snapshots (via Microlink API) of live Vercel deployments.
- **Dynamic Skill Icons**: Auto-maps technologies to their respective Devicon / JSON logos.

## 🛠️ Tech Stack

- **Framework**: React.js 18 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Icons**: Lucide React + Devicon
- **Deployment**: Vercel

## 📂 Project Structure

- `src/components`: Reusable UI elements (Navigation, CursorTrail, StarryBackground).
- `src/sections`: Major page sections (Home, Skills, Work, Experience, Contact).
- `src/data`: Contains the `resumeData.ts` file which statically serves all the dynamic content (Projects, Experience, Skills, Highlights).
- `public`: Static assets, SVGs, and specific local project screenshots (`attendance.png`, `enviro.png`).

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and NPM/Yarn installed.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/jaswanth1904/Jaswanth-s-portfolio.git
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Run the development server
   ```bash
   npm run dev
   ```

4. Build for production
   ```bash
   npm run build
   ```
